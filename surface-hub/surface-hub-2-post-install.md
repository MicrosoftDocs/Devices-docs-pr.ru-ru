# <a name="configure-windows-10-pro-or-enterprise-on-surface-hub-2"></a>Настройка Windows 10 Pro или Корпоративная на Surface Hub 2

После завершения процесса установки перехода на Windows 10 Pro или Enterprise можно выполнить следующие действия по настройке приложений и параметров на surface Hub 2. Эти действия рекомендуется для обеспечения наилучшего опыта при использовании этого персонализированного большого сенсорного экрана и пера компьютера.

При выполнении этих действий может оказаться полезным использовать проводную или беспроводную клавиатуру и мышь.

## <a name="configure-system-settings"></a>Настройка параметров системы

1. Во входе в учетную запись с локальными привилегиями администратора на устройстве.  

    - На присоединив устройствах Azure AD пользователь, который выполняет присоединиться к Azure AD, автоматически добавляется в локализованную группу администраторов. Глобальные администраторы Azure AD и администраторы устройств Azure AD также <a href="https://docs.microsoft.com/azure/active-directory/devices/assign-local-admin" target="_blank"> являются локальными администраторами. </a> 
    
    - Вы можете ввести **чистых администраторов** локальных групп по командной подсказке, чтобы перечислить учетные записи, которые имеют права локального администратора.
    
2. Переименование устройства с использованием удобного имени, например: **username-SHub-Desktop.**

3. Выберите ****  >  **Учетные записи параметров**начните  >  ****  >  **синхронизировать параметры** и отключить **параметры Синхронизации.** 

    - Параметры, используемые здесь, предназначены для обеспечения наилучшего сенсорного экрана на большом экране, поэтому вам может не потребоваться синхронизировать другие устройства.
    
4. Перезагрузите устройство.

## <a name="enable-the-touch-keyboard-and-touchpad"></a>Включить сенсорную клавиатуру и сенсорную панель

1. Выберите **Начните**параметров устройств ввода и включить Показать сенсорную клавиатуру, когда не в режиме планшета и нет клавиатуры,  >  ****  >  ****  >  **** **присоединенной.**

2. Нажмите и удерживайте или нажмите правой кнопкой мыши на панели задач, а затем выберите **кнопку Показать** сенсорную клавиатуру и **кнопку Show touchpad**. 

    - Сенсорная клавиатура полезна для прямого ввода пользователя, а виртуальная сенсорная панель помогает с точными выборами, наведении на экране подсказки или в качестве альтернативы нажатию и удержанию правой кнопкой мыши. 
    
    - См. следующий пример.

      ![Параметры касания](images/touch.png)

3. Настройка сенсорной клавиатуры на QWERTY и плавающей.

    1. Выберите **значок Клавиатура** на панели задач, чтобы показать сенсорную клавиатуру.
    
    1. На сенсорной клавиатуре выберите значок клавиатуры в левом верхнем углу, чтобы открыть параметры клавиатуры.
    
    1. Выберите следующий тип клавиатуры в верхней строке, чтобы включить QWERTY, а последний вариант на втором ряду для обеспечения плавающего, что очень полезно на этом большом экране. См. следующие примеры.

       ![Параметры клавиатуры](images/kbd.png)
 
4. Настройка параметров мягкой клавиатуры.

    1. Выберите **значок Параметры** на сенсорной клавиатуре или наведите и откройте **параметры ввода.**
    
       ![параметры мягкой клавиатуры](images/sh2-softkeyboard.png)

    1. Введите все параметры в клавиатуре Spelling, Typing и Touch.


В следующем примере показан трекпад, который полезен для навигации и выбора параметров. Экранная клавиатура используется для поиска в Microsoft Store:

## <a name="onedrive-for-business"></a>OneDrive для бизнеса

Используйте <a href="https://docs.microsoft.com/onedrive/onedrive" target="_blank"> OneDrive для бизнеса, чтобы легко обмениваться инструментами, журналами и другими </a> файлами между всеми вашими устройствами.

- OneDrive позволяет обмениваться рабочими файлами между ноутбуками, настольными устройствами Surface Hub и мобильными устройствами с управлением Intune. Файлы можно редактировать на любом устройстве, и все подключенные к сети устройства будут обновлены с помощью изменений.

- Учитывая размер SSD Surface Hub (128 ГБ), если вы настраиваете OneDrive на рабочем столе Surface Hub, убедитесь, что конфигурация по умолчанию должна хранить файлы в Интернете и скачивать файлы при их использовании.

Чтобы настроить OneDrive для загрузки файлов только при необходимости, установите параметр **Files On-Demand** для сохранения пространства и загрузки файлов при **их использовании.** Дополнительные сведения см. в <a href="https://docs.microsoft.com/onedrive/files-on-demand-windows" target="_blank"> материалах "Запрос" и "Настройка состояния файлов по требованию" в </a> Windows.

![Параметры OneDrive](images/onedrive.png)

> [!NOTE]
> Вы также можете повторить эти действия, чтобы настроить личный OneDrive, но не забудьте сохранить пространство диска и скачивать файлы только по мере их необходимости.

## <a name="sharepoint-and-teams"></a>SharePoint и Teams

Файлы каналов SharePoint и Teams также могут локально синхронизироваться с настольными устройствами, такими как ноутбуки и концентраторы Surface, с помощью двигателя синхронизации OneDrive.

Синхронизация внутренних корпоративных файлов с локальным диском с приложением синхронизации OneDrive:

1. Перейдите на сайт SharePoint и перейдите в каталог документов верхнего уровня для файлов, которые вам интересны для просмотра или редактирования с локального устройства.

2. Выберите **кнопку Sync** в верхней части ленты SharePoint.

3. Выберите в **open** в **всплывающее всплывающее место Этот**сайт пытается открыть Microsoft OneDrive .

4. Убедитесь, что файлы SharePoint синхронизируются с локальным диском, выбрав значок OneDrive в правом нижнем справа от панели задач.

5. Убедитесь, что конфигурация установлена для сохраняемой онлайн-загрузки файлов только при их использовании:

    1. Откройте проводник файлов.
    
    2. Перейдите вправо и вправо щелкните имя SharePoint; например, **Contoso \<SharePoint Document Folder Name\> \ **.
    
    3. Выберите **свободное пространство.**
    
    4. Столбец Status отображает состояние файлов и папок. Дополнительные сведения см. в <a href="https://support.microsoft.com/office/sync-sharepoint-files-with-the-onedrive-sync-client-groove-exe-59b1de2b-519e-4d3a-8f45-51647cf291cd" target="_blank"> раздел Синхронизация файлов SharePoint с клиентом синхронизации OneDrive. </a>
    
6. Файлы каналов Teams хранятся на сайтах SharePoint с одинаковыми функциями документов SharePoint, включая историю версий и синхронизацию с локальными настольными устройствами. Синхронизация файлов каналов Teams:

    1. Перейдите к интересуемого канала Teams и выберите вкладку **Files** в верхней части. Затем выберите **Синхронизация**. Файлы начнут синхронизироваться и будут видны в Проводнике файлов на рабочем столе **\ Contoso \<name of the Teams Channel\> \ **.
    
    2. Используйте ту же процедуру, что и для синхронизации сайтов SharePoint, чтобы хранить файлы в облаке и загружать их только при их использовании, нажав на кнопку и удерживая или щелкнув правой кнопкой мыши в проводнике файлов в имени канала teams, а затем выбрав свободное пространство **.**

## <a name="camera-configuration"></a>Конфигурация камеры

Вы можете установить камеру на верхней или с обеих сторон устройства. Смонтируйте камеру в положение, чтобы оптимизировать угол камеры, если вы используете концентратор с настольной стойкой вместо корзины или находитесь в непосредственной близости от концентратора. Камера не автоматически вращается, поэтому для ручного поворота камеры необходимо иметь 2-миллиметровую гексную клавишу. 

Дополнительные сведения о том, как смонтировать камеру и вращать ее вручную, см. в инструкции по ориентации объектива <a href="https://support.microsoft.com/help/4509729/surface-hub-2s-camera-lens-orientation" target="_blank"> камеры Surface Hub 2S. </a>

Surface Hub 2S под управлением Windows 10 Enterprise позволяет использовать полный набор настольных приложений Win32, а также биометрические параметры Windows Hello. Аксессуар Surface Hub 2 Fingerprint Reader можно подключить к любому порту USB-C на устройстве. 

Чтобы заказать сканер отпечатков пальцев Surface Hub 2 или просмотреть технические спецификации, см. (surface-hub-2-essential-add-ons.md" target="_blank">Основные надстройки для Windows 10 Pro и Enterprise на Surface Hub 2 </a> . 

После вставки сканера **** отпечатков пальцев выберите параметры входных параметров учетных записей Параметров Параметров параметров  >  ****  >  ****  >  ****  >  **Windows Hello Fingerprint** для регистрации отпечатков пальцев.

Используйте сертифицированное устройство Windows Hello для распознавания лиц. Камера Surface Hub 2S не поддерживает распознавание лиц Windows Hello.

###  <a name="fancy-zones"></a>Необычные зоны


**Fancy Zones** — это часть коллекции инструментов под названием <a href="https://github.com/microsoft/PowerToys/releases" target="_blank"> PowerToys </a> на GitHub. Это отличный способ использовать экранную недвижимость на Surface Hub 2, дав вам возможность определять фиксированные макеты на дисплее ("зоны"), а затем выбрать, какое приложение будет работать в каждой зоне. 


В [вики PowerToys](https://github.com/microsoft/PowerToys/wiki) есть инструкции по использованию и настройке каждого средства, включая [FancyZones.](https://github.com/microsoft/PowerToys/wiki/FancyZones-Overview) На высоком уровне после установки PowerToys можно выбрать или создать настраиваемый макет, а затем удерживать клавишу переноса вниз и перетаскивать или использовать клавиши клавиатуры для перемещения запущенного приложения в определенные зоны. С помощью Bluetooth или USB-клавиатуры и мыши это поможет, или вы можете использовать сенсорную клавиатуру и сенсорную панель на экране.

**Power toys tips**
- Чтобы получать уведомления о выпуске обновлений PowerToys в GitHub, щелкните кнопку "вход" в верхней части [страницы.](https://github.com/microsoft/PowerToys/releases)
- После установки PowerToys вы можете получать уведомления Windows и/или загружать и устанавливать последние обновления, **** настраивая параметры PowerToys Для автоматического скачивания обновлений.
- Чтобы добраться до параметров PowerToys, **** выберите вверх карат Запуск приложений на панели задач, а затем правой кнопкой мыши или нажмите кнопку и удерживайте значок PowerToys до тех пор, пока меню не появится. Выберите параметры.
- В нижней части страницы параметров PowerToys автоматически **включите** обновления загрузки.
- После выпуска обновления появится уведомление о том, когда необходимо установить обновление.

### <a name="power-management"></a>Управление питанием

Существует несколько параметров питания, которые можно получить с помощью Windows 10 Pro или Enterprise на Surface Hub 2. Это включает в себя время отключения экрана и пк и их взаимодействие со встроенным обнаружением присутствия человека (Doppler), средствами защиты от сохранения экрана и паролем, а затем, если это уместно, чтобы передать параметры питания групповой политики, предназначенные для пользователей ноутбуков и настольных компьютеров.

Windows 10 Pro или Enterprise на Surface Hub 2 не позволяет экрану спать при сенсорном, мыши и клавиатуре, а также при встроенном обнаружении заполняемости человека (Doppler). Обнаружение заполняемости человека включено по умолчанию, но при желании его можно отключить в UEFI, переключив параметр устройства в средство Конфигуратора Surface UEFI либо в рамках первоначальной миграции, либо путем создания и применения более позднего пакета конфигурации UEFI. 

**Управление питанием: параметры сна на экране и ПК**

1. Выберите ****  >  **Начните**  >  **параметры**  >  **системной & сна.**

2. Установите ползунок режима питания на **лучшую производительность.**

3. Настройка значений экрана и сна в ваших предпочтениях, а также учет обнаружения присутствия Doppler, которое пробуждает устройство при обнаружении движения. Соответственно, в качестве наилучшей практики рекомендуется настроить отключение экрана через **2** часа, а компьютер отключить **через 4 часа.**

**Управление питанием: за исключением экрана**

1. Поиск **параметров экрана блокировки** и **открытых параметров экрана блокировки.**

2. Настройка **параметров времени сохранения экрана** и **параметров** сохранения экрана в ваших предпочтениях. Рекомендуемые значения по умолчанию:

   - За исключением экрана (Нет) или за исключением экрана по вашему выбору.
   - Время ожидания до 15 минут.
   - В резюме отобразить экран логотипа.


**Управление питанием: групповой политики**

Перед выполнением следующей процедуры обратитесь в ИТ-отдел для утверждения, чтобы исключить устройство Surface Hub 2S из глобальной политики управления питанием. Некоторые параметры управления питанием могут отключить функцию обнаружения присутствия.

1. Поиск центра **программного обеспечения и** его открытие.

2. Выбор **параметров**.

3. **Расширь управление power и** выберите Не применяй параметры питания из моего **ИТ-отдела на этот компьютер.**

   ![Параметры программного обеспечения](images/soft-cntr.png)

### <a name="storage-sense"></a>Контроль памяти

Surface Hub 2 имеет SSD объемом 128 ГБ для локального хранения, поэтому необходимо учитывать использование мер по экономии хранилища во время нормального использования.  Чтобы настроить чувство хранилища:

1.  Поиск **параметров хранения,** которые находятся в **параметрах System.**

2.  В **параметрах**выберите **включив чувство хранилища,** чтобы открыть страницу **параметры** хранилища.

3.  Включи чувство **хранения в включить**.

4.  Выберите **Настройте чувство хранилища или** запустите его сейчас и настройте параметры, чтобы держать файлы в сети как можно больше (из-за ограниченного пространства диска).

Рекомендуемые параметры:

- Запустите чувство хранилища = каждый день.
- Удаление временных файлов, которые не используют мои приложения = Каждые 14 дней (по крайней мере).
- Удаление файлов в папке Загрузки, если они были там более = 30 дней.
- OneDrive: Контент станет только в Интернете, если не будет открыт более = 30 дней.

## <a name="complete-and-verify"></a>Завершение и проверка

1. Сканирование и установка всех обновлений Windows.

2. Обновление групповой политики.

   1. При повышенной командной подсказке введите **gpupdate /force/boot/wait:0**.
   
3. Перезагрузите устройство.

4. Проверка приложений панели задач.

   - Подключение приложения
   - Значок блокировки
   - Набросок на фрагменте экрана
   - Teams (если применимо)
   - Приложения Office (если применимо)
   - Surface App
   - Доска
    
5. Проверка обнаружения присутствия.

   - Обнаружение присутствия будет зеленым значком в подносе системы.
    
6. Проверка включения проеции на этот компьютер с помощью приложения Connect. После настройки  **параметров Project на этом компьютере** запустите приложение Connect По крайней мере один раз. (Впоследствии приложение Connect не нужно запускать, чтобы проектов в Surface Hub.)

7. Проверка параметров питания и сна.

    - Сохранение экрана: 15 минут, задав (нет), Mystify или Blank; убедитесь, что выбрано поле для запроса пароля.
    - Экран: **отключить через 2 часа**.
    - PC. **Выключите через 4 часа.**
    
8. Проверка работы Windows Hello.

9. Убедитесь, что синхронизация параметров отключена.

10. Проверка запуска приложений.
