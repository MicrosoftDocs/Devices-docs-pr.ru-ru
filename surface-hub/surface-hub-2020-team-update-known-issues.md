# Известные проблемы: Windows 10 Team 2020 обновление Предварительная сборка 

На этой странице содержится список известных проблем, связанных с сборкой предварительной версии Windows 10 Team 2020 Update. Проблемы, исправленные с момента выпуска предварительной сборки, перечислены в таблице 2. 

#### Таблица1. Известные проблемы — сборка предварительной версии Windows 10 Team 2020 Update 

| Проблема                                                                     | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| ------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Проблемы с графикой, звуком и интерфейсом пользователя**                            | При установке предварительной сборки вы можете столкнуться с различными неполадками в графике и пользовательском интерфейсе, но вы не сможете сразу же установить необходимые обновления встроенного по позже. Корпорация Майкрософт планирует устранить эти проблемы в выпуске обновления для Windows 10 Team 2020.                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| **Вход имеет влияние на то, что применяются политики условного доступа**      | -При попытке входа в приложения, такие как Microsoft доска, вход завершается сбоем. Пользователи должны сначала войти в меню "Пуск" на [странице "Мои собрания и файлы](https://support.microsoft.com/help/4506480/sign-in-to-see-your-meetings-and-files-on-surface-hub) ".<br>-Вход завершается сбоем, если ваша учетная запись пользователя зарегистрирована в другом клиенте Azure AD, чем тот, который используется в качестве Surface Hub для Azure AD JOIN.<br><br>Корпорация Майкрософт планирует устранить эти проблемы в выпуске обновления для Windows 10 Team 2020.                                                                                                                                                                                              |
| **Windows Update предлагает установить новые драйверы, если их нет в наличии** | После установки **обновления Windows**  >  **Update & Security**  >  **Windows Update** 10 Team 2020, а также необходимых обновлений встроенного по, вы можете увидеть следующее сообщение об ошибке: &<br><br>-"Текущий драйвер на компьютере может быть лучше, чем драйвер, который вы пытаетесь установить. Мы будем продолжать попытки установить ".<br><br>Эту ошибку можно игнорировать. Корпорация Майкрософт активно изучает эту проблему.                                                                                                                                                                                                                                                                              |
| **Запрашивать удаление USB-накопителя преждевременно при первом запуске**               | При использовании пакета подготовки для настройки Surface Hub во время первого запуска вам будет предложено удалить USB-накопитель, прежде чем устройство сможет прочитать файл конфигурации Surface Hub. Если вы используете файл конфигурации для настройки учетной записи устройства, пропустите сообщение. Корпорация Майкрософт активно изучает эту проблему.                                                                                                                                                                                                                                                                                                                                                                          |
| **При загрузке приложений из Microsoft Store появляется сообщение об ошибке**          | Чтобы загрузить приложение из Microsoft Store, вы увидите запрос на вход. Известная проблема приводит к появлению ошибки при входе, но это не повлияет на возможность скачивания приложений. Корпорация Майкрософт активно изучает эту проблему.                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| **Surface Hub 2 периодически теряет соединение Wi-Fi**                  | Попробуйте отключить устройство и снова подключить его, а также с помощью кабеля Ethernet для подключения к Интернету. Корпорация Майкрософт активно изучает эту проблему.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| **Периодически не удается возобновить работу из спящего режима на Surface Hub 2**              | Surface Hub 2 иногда не смогут полностью выйти из спящего режима и перестанет отвечать на экран с логотипом Microsoft. Попробуйте отключить устройство и снова подключить его.<br><br>**Чтобы избежать этой ошибки, выполните указанные ниже действия.**<br><br>1. на Surface Hub откройте **Параметры** и введите свои учетные данные администратора при появлении соответствующего запроса.<br>2. Перейдите к сеансу **Surface Hub**  >  **& Power**.<br>3. в разделе **при переходе устройства в спящий режим используйте этот параметр питания**и выберите **режим ожидания с подключением.**<br>4. в разделе **когда никто не будет находиться в спящем режиме**, нажмите кнопку **Never (никогда).**<br><br>Корпорация Майкрософт планирует устранить эту проблему в обновлении встроенного по до окончательного выпуска обновления для Windows 10 Team 2020. |
