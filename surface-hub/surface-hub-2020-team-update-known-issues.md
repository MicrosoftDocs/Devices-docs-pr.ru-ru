# Известные проблемы: Обновление Windows 10 для группы 2020 

В этой статье перечислены известные проблемы для Windows 10 Team 2020 Update, текущей операционной системы для Surface Hub.

Чтобы обеспечить получение Surface Hub последних обновлений, во sign in with an Admin account and select **All apps**  >  **Settings**  >  **Update and Security**Windows  >  **Update**, and then install all updates.

В результате такой конфигурации прокси-сервер не определяет учетную запись устройства как допустимого пользователя для проверки подлинности и отклоняет запрос.
