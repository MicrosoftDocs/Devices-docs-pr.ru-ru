# Локальное управление параметрами Surface Hub

После первоначальной настройки Microsoft Surface Hub параметрами устройства можно управлять локально с помощью приложения **Параметры**.

## Параметры Surface Hub

У устройства Surface Hub существует много параметров, которые также есть и на других устройствах с Windows, но ряд параметров доступны только на Surface Hub. В этой таблице перечислены параметры, которые можно настроить только в Surface Hub. 

| Параметр | Расположение | Описание |
| ------- | -------- | ----------- |
| Учетная запись устройства | Surface Hub > Учетные записи | Настройка или изменение учетной записи устройства Surface Hub. |
| Состояние синхронизации учетной записи устройства | Surface Hub > Учетные записи | Проверка состояния синхронизации почты и календаря учетной записи устройства Surface Hub. |
| Чередование паролей | Surface Hub > Учетные записи | Выберите, нужно ли разрешить Surface Hub автоматически менять пароль учетной записи устройства.|
| Изменить пароль учетной записи администратора  | Surface Hub > Учетные записи | Изменения пароля для локальной учетной записи администратора. Эта функция доступна, только если устройство настроено для использования локального администратора во время первого запуска. |
| Управление устройствами | Surface Hub > Управление устройства | Управление политиками и бизнес-приложениями с помощью управления мобильными устройствами (MDM). |
| Пакеты подготовки | Surface Hub > Управление устройства | Установка или изменение пакетов подготовки, установленных на устройстве Surface Hub. |
| Открытие приложения Microsoft Store | Surface Hub > Приложения и компоненты | Приложение Microsoft Store доступно только для администраторов через приложение "Параметры". |
| Доменное имя Skype для бизнеса | Surface Hub > Вызовы и звук | Настройка доменного имени Skype для бизнеса Server. |
| Громкость динамиков по умолчанию | Surface Hub > Вызовы и звук | Настройка звука динамиков по умолчанию для Surface Hub при запуске сеанса. |
| Параметры микрофона и динамика по умолчанию | Surface Hub > Вызовы и звук | Настройка по умолчанию микрофона и динамика для вызовов и динамика по умолчанию для воспроизведения мультимедиа. |
| Включение звука Dolby X2 | Surface Hub > Вызовы и звук | Настройка улучшений звука Dolby X2 для динамиков. |
| Автоматическое открытие приложения Connect | Surface Hub > Проецирование | Выберите, будет ли при проектировании автоматически открываться приложение Connect или будут ожидаться действия пользователя. |
| Отключить беспроводную проекцию с помощью Miracast | Surface Hub > Проецирование | Выберите, могут ли выступающие проецировать презентацию по беспроводной сети на устройство Surface Hub с помощью Miracast. |
| Требовать PIN-код для беспроводной проекции | Surface Hub > Проецирование | Выберите, требуется ли пользователям вводить PIN-код для беспроводной проекции. |
| Канал беспроводной проекции (Miracast) | Surface Hub > Проецирование | Настройка канала проекции Miracast. |
| Сведения о собрании, отображаемые на экране приветствия | Surface Hub > Экран приветствия | Выберите, будут ли организатор, время и тема собрания отображаться на экране приветствия. |
| Фон экрана приветствия |  Surface Hub > Экран приветствия | Выберите изображение фона для экрана приветствия. |
| Время ожидания сеанса на экране приветствия | Surface Hub > сеанс & Power | Укажите, через какое время с момента, когда перестало обнаруживаться движение, будет появляться экран приветствия на Surface Hub. |
| Возобновление сеанса | Surface Hub > сеанс & Power | Выберите, нужно ли разрешить пользователям восстанавливать сеанс, после того как прекратиться обнаружение движения, или автоматически очищать сеанс. |
| Доступ к собраниям и файлам Office 365 | Surface Hub > сеанс & Power | Выберите, сможет ли пользователь входить в Office 365, чтобы получать доступ к своим собраниям и файлам. |
| Включать экран датчиками движения | Surface Hub > сеанс & Power | Выберите, включается ли экран при обнаружении движения. |
| Время ожидания для экрана | Surface Hub > сеанс & Power | Укажите, как долго устройство должно быть неактивным, прежде чем отключить экран. |
| Тайм-аут спящего режима | Surface Hub > сеанс & Power | Выберите длительность периода неактивности, после которого устройство перейдет в режим сна. |
| Понятное имя | Surface Hub > Сведения | Изменение имени Surface Hub, которое пользователи будут видеть при беспроводном подключении к нему. |
| Часы обслуживания | Обновления и безопасность > Центр обновления Windows > Дополнительные параметры | Настройка времени установки обновлений. |
| Настроить сервер служб Windows Server Update Services (WSUS) | Обновления и безопасность > Центр обновления Windows > Дополнительные параметры | Измените способ получения Surface Hub обновлений: с сервера WSUS вместо Центра обновления Windows. |
| Восстановление из облака | Обновление и безопасность > Восстановление. | Переустановите операционную систему на Surface Hub, восстановив сборку изготовителя из облака. |
| Сохранить ключ BitLocker | Обновление и безопасность > Восстановление. | Резервное копирование ключа BitLocker Surface Hub на USB-накопитель. |
| Собирать журналы | Обновление и безопасность > Восстановление. | Журналы можно сохранить на USB-накопитель и отправить в корпорацию Майкрософт позже. |
