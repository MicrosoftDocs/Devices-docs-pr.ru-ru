# Развертывание набора средств диагностики Surface для бизнеса

Набор средств Microsoft Surface Diagnostic Toolkit для бизнеса (SDT) позволяет ИТ-администраторам быстро находить, устранять проблемы с оборудованием, программным обеспечением и встроенными микрокодом с помощью устройств Surface. Вы можете выполнять диапазон диагностических тестов и восстановление программного обеспечения в дополнение к получению сведений о работоспособности устройств и руководство по устранению проблем. 

В частности, SDT для бизнеса позволяет:

- [Настройте пакет.](#create-custom-sdt)
- [Запустите приложение с помощью команд.](surface-diagnostic-toolkit-command-line.md)
- [Запустите несколько тестов оборудования, чтобы устранить неполадки.](surface-diagnostic-toolkit-desktop-mode.md#multiple)
- [Создание журналов для анализа проблем.](surface-diagnostic-toolkit-desktop-mode.md#logs)
- [Получите подробный отчет о сравнении устройств и оптимальной конфигурации.](surface-diagnostic-toolkit-desktop-mode.md#detailed-report)

## Установка набора средств диагностики Surface для бизнеса

Чтобы создать пакет SDT, который вы можете распространять пользователям в своей организации, выполните указанные ниже действия.

1. Войдите на устройство Surface с помощью учетной записи администратора.
2. Скачайте пакет установщика Windows (MSI-файл) SDT на [странице Surface Tools for IT](https://www.microsoft.com/download/details.aspx?id=46703) и скопируйте его в нужное место на устройстве Surface, например на компьютере.
3. Откроется мастер настройки SDT, как показано на рисунке 1. Нажмите кнопку **Далее**. 

    >[!NOTE]
    >Если мастер настройки не отображается, убедитесь в том, что вы вошли в учетную запись администратора на вашем компьютере. 

    ![Добро пожаловать в мастер настройки средства диагностики Surface Toolkit](images/sdt-1.png)

    *Рисунок 1. Мастер настройки набора средств диагностики Surface*

4. Когда появится мастер настройки SDT, нажмите кнопку **Далее**, приняв условия лицензионного соглашения конечного пользователя (EULA)

5. На экране параметры установки измените расположение для установки по умолчанию (при необходимости). 
6. В разделе Тип настройки выберите пункт **Дополнительно**. 

    >[!NOTE]
    >Стандартный параметр позволяет пользователям запускать средство диагностики прямо на устройстве Surface, если они вошли в систему с помощью учетной записи администратора. 
    
     ![Варианты установки: дополнительно](images/sdt-install.png)

7. Нажмите кнопку **Далее** , а затем — **установить**. 

## Установка с помощью командной строки
При необходимости вы можете установить SDT в командной строке и установить настраиваемый флажок, чтобы установить средство в режиме администратора. SDT включает следующие флаги параметров установки:

- `SENDTELEMETRY` Отправка телеметрических данных в корпорацию Майкрософт. Этот флаг принимает состояние " `0` Отключено" или " `1` включено". Значение по умолчанию — `1` Отправить телеметрию.
- `ADMINMODE` Настройка установки средства в режиме администратора. Этот флаг принимается `0` для клиентского режима или `1` режима ИТ администратора. Значение по умолчанию— `0`.

### Чтобы установить SDT из командной строки, выполните указанные ниже действия.

1. Откройте командную команду и введите:

    ```
    msiexec.exe /i <the path of installer> ADMINMODE=1. 
    ```
    **Пример.**

    ```
    C:\Users\Administrator> msiexec.exe/I"C:\Users\Administrator\Desktop\Microsoft_Surface_Diagnostic_Toolkit_for_Business_Installer.msi" ADMINMODE=1
    ```

## Подготовка пакета SDT для распространения

Создание настраиваемого пакета позволяет нацелить средство на конкретные известные проблемы.

1. Нажмите кнопку **пуск > выполнить**, введите **Surface** и щелкните **набор средств диагностики Surface для бизнеса**. 
2. Когда откроется средство, нажмите **создать настраиваемый пакет**, как показано на рис. 3.

    ![Параметр "создать пользовательский пакет"](images/sdt-3.png)

    *Рисунок 3. Создание настраиваемого пакета*

### Параметры языка и телеметрии

  При создании пакета вы можете выбрать языковые параметры или отказаться от отправки сведений о телеметрии в корпорацию Майкрософт. По умолчанию SDT отправляет телеметрию в корпорацию Майкрософт, которая используется для улучшения приложения в соответствии с заявлением [о конфиденциальности корпорации Майкрософт](https://privacy.microsoft.com/privacystatement). Если вы хотите отклонить, снимите флажок при создании настраиваемого пакета, как показано ниже. Или снимите флажок **Отправить телеметрию в Microsoft** на странице **параметров установки** во время настройки SDT. 

>[!NOTE]
>Этот параметр не влияет на минимальную телеметрию, которая автоматически хранится на серверах Microsoft при выполнении тестов и исправлений, требующих подключения к Интернету, таких как обновление Windows и восстановление программного обеспечения, и Отправка отзывов с помощью кнопок улыбки и frown на панели инструментов приложения.
